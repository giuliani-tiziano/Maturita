Topic 01 · Computing Fundamentals

Computer Hardware & Architecture

CPU · RAM · Storage · Motherboard · Von Neumann Architecture · Moore's Law

Overview

What is computer hardware?

Hardware refers to the physical components of a computer system — the parts you can touch. It is distinguished from software (programs and operating systems) and firmware (software embedded in hardware). Understanding hardware is fundamental to all areas of IT: networking, cybersecurity, programming and systems administration all depend on knowing how the underlying machine works.

The five core components

  1. CPU (Central Processing Unit) — the processor: executes instructions.
  2. RAM (Random Access Memory) — volatile working memory.
  3. Storage — persistent memory: HDD, SSD, NVMe.
  4. Motherboard — the main circuit board connecting all components.
  5. I/O devices — input (keyboard, mouse) and output (monitor, printer).

Von Neumann Architecture (1945)

Proposed by John von Neumann, this architecture — still the basis of modern computers — has four units:

  • ALU — Arithmetic Logic Unit: does calculations.
  • Control Unit — fetches and decodes instructions.
  • Memory — stores both data and instructions in the same space.
  • I/O — communicates with the outside world.

The key feature: data and programs share the same memory — this is the "stored program" concept.

Key Concept

The CPU — fetch, decode, execute

The CPU operates in a continuous cycle called the fetch-decode-execute cycle:

  1. Fetch — the Control Unit retrieves an instruction from RAM.
  2. Decode — the instruction is interpreted: what operation? what data?
  3. Execute — the ALU performs the operation.
  4. Store — the result is written back to RAM or a register.

Speed is measured in GHz (gigahertz = billion cycles per second). A modern CPU performs billions of these cycles every second.

Moore's Law

In 1965, Intel co-founder Gordon Moore observed that the number of transistors on a chip doubles approximately every two years, while costs fall. This "law" (actually an observation) drove exponential growth in computing power for 50 years.

Today Moore's Law is slowing: transistors are reaching the physical limits of silicon. The industry is exploring new approaches: 3D chip stacking, quantum computing, neuromorphic chips.
