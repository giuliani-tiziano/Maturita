Tech & Digital English

20 Topics

From hardware architecture to emerging technologies — complete in English

Computing Fundamentals

Topics 01–05

01
Computer Hardware & Architecture
CPU, RAM, storage, motherboard, buses. Von Neumann architecture vs Harvard architecture. Moore's Law.

Key vocabulary

CPU (Central Processing Unit) — the "brain" of the computer. RAM (Random Access Memory) — volatile, fast temporary storage. ROM — non-volatile permanent memory. Bus — a communication channel between components. Clock speed — measured in GHz, the speed at which the CPU executes instructions. Cache — ultra-fast memory close to the CPU.
02
Operating Systems
Functions: process management, memory management, file management, I/O control. Types: single-user, multi-user, real-time, distributed. Linux, Windows, macOS.

Key vocabulary

Kernel — the core of an OS, mediating between hardware and software. Process — a running program. Thread — the smallest unit of execution within a process. Multitasking — running multiple processes "simultaneously." File system — the structure for organising data on disk (FAT32, NTFS, ext4).
03
Data Representation
Binary, hexadecimal, ASCII, Unicode. How images (pixels, RGB), audio (sampling, bitrate), video are digitised. Compression (lossless vs lossy).

Key vocabulary

Bit — binary digit (0 or 1). Byte — 8 bits. ASCII — 7-bit encoding for 128 characters. Unicode — universal encoding supporting all world scripts. Lossless — compression with no data loss (ZIP, PNG). Lossy — compression that removes data (JPEG, MP3).

04
Algorithms & Flowcharts
Algorithm definition, pseudocode, flowchart symbols. Sorting algorithms (bubble, selection, merge). Searching algorithms (linear, binary). Complexity O notation.

Key vocabulary

Algorithm — a step-by-step procedure to solve a problem. Flowchart — a visual representation of an algorithm. Time complexity — how execution time grows with input size. O(n) — linear complexity. O(n²) — quadratic complexity (inefficient). Recursion — a function that calls itself.

05
Programming Languages
Low-level vs high-level. Compiled vs interpreted. Procedural, OOP, functional paradigms. History: FORTRAN → C → Java → Python → modern languages.

Key vocabulary

Compiler — translates entire source code to machine code before execution. Interpreter — translates and executes line by line. Syntax — the rules governing the structure of valid code. Semantics — the meaning of code. Variable — a named storage location. Function — a reusable block of code.

Software Development

Topics 06–07

06
Object-Oriented Programming
Classes, objects, encapsulation, inheritance, polymorphism, abstraction. SOLID principles. Java, Python, C++ as OOP languages.

Key vocabulary

Class — a blueprint for creating objects. Object — an instance of a class. Encapsulation — hiding internal implementation. Inheritance — a class acquiring properties of another. Polymorphism — same interface, different implementation. Constructor — a method that initialises an object.

07
Databases & SQL
Relational databases, tables, primary/foreign keys. SQL: SELECT, INSERT, UPDATE, DELETE. NoSQL databases. ERD diagrams. ACID properties.

Key vocabulary

DBMS — Database Management System. Table — data organised in rows (records) and columns (fields). Primary key — unique identifier for each record. Foreign key — a reference to a primary key in another table. Query — a request to retrieve or manipulate data. Transaction — a unit of database work (all-or-nothing).

Networking & Internet

Topics 08–10

08
Computer Networks
LAN, WAN, MAN. Topologies: star, ring, bus, mesh. OSI model (7 layers). TCP/IP model. Protocols: Ethernet, Wi-Fi. Switches, routers, hubs.

Key vocabulary

Protocol — an agreed set of rules for communication. IP address — a numerical label identifying a network device. MAC address — hardware address unique to each network interface. Router — forwards packets between networks. Bandwidth — the maximum data transfer rate. Latency — delay in data transmission.
09
The Internet & Protocols
Internet history (ARPANET 1969). TCP/IP, DNS, DHCP, HTTP/HTTPS, FTP, SMTP. IPv4 vs IPv6. Packet switching. ISPs.

Key vocabulary

DNS — Domain Name System: translates domain names to IP addresses. TCP — Transmission Control Protocol: reliable, ordered delivery. UDP — faster, unreliable protocol for streaming. Packet — a unit of data transmitted over a network. Port — a virtual endpoint for network communication (e.g., port 80 = HTTP).
10
World Wide Web & HTTP
WWW invented by Tim Berners-Lee (1989). HTTP vs HTTPS. URLs, HTML, CSS, JavaScript. Client-server model. Cookies, sessions, REST APIs.

Key vocabulary

URL — Uniform Resource Locator: the address of a web resource. HTTP — HyperText Transfer Protocol. SSL/TLS — encryption protocols for HTTPS. Browser — software that retrieves and renders web pages. Server — a computer that provides services to clients. API — Application Programming Interface.

Security

Topics 11–12

11
Cybersecurity & Threats
Malware types (virus, worm, trojan, ransomware, spyware). Social engineering: phishing, pretexting. CIA triad (Confidentiality, Integrity, Availability). Firewalls, antivirus, VPN.

Key vocabulary

Malware — malicious software. Phishing — fraudulent emails/websites to steal credentials. Ransomware — encrypts data and demands payment. DDoS — Distributed Denial of Service attack. Firewall — filters network traffic. Two-factor authentication — requires two forms of verification.
12
Cryptography
Symmetric vs asymmetric encryption. RSA, AES. Public key infrastructure. Digital signatures and certificates. Hashing (SHA, MD5). End-to-end encryption.

Key vocabulary

Encryption — converting plaintext to ciphertext using a key. Symmetric — same key for encryption and decryption (AES). Asymmetric — public key encrypts, private key decrypts (RSA). Hash — a fixed-length fingerprint of data (SHA-256). Digital certificate — links a public key to an identity. HTTPS — HTTP with TLS encryption.

Emerging Technologies

Topics 13–16

13
Cloud Computing
IaaS, PaaS, SaaS. Public, private, hybrid cloud. Providers: AWS, Azure, Google Cloud. Virtualisation, containers (Docker), microservices. Scalability, elasticity.

Key vocabulary

Cloud — IT resources delivered over the internet. SaaS — Software as a Service (Gmail, Office 365). IaaS — Infrastructure as a Service (virtual machines). PaaS — Platform as a Service (Heroku, App Engine). Scalability — ability to handle growing workloads. Virtualisation — running multiple OS on one physical machine.
14
Artificial Intelligence & Machine Learning
AI history (Turing Test 1950). ML types: supervised, unsupervised, reinforcement. Neural networks, deep learning. Natural language processing. Ethical concerns: bias, accountability.

Key vocabulary

AI — Artificial Intelligence: machines simulating human intelligence. ML — Machine Learning: systems that learn from data. Neural network — computational model inspired by the brain. Training data — the dataset used to train a model. Algorithm bias — systematic errors in AI due to biased training data. NLP — Natural Language Processing.
15
Internet of Things (IoT)
Connected everyday objects. Smart home, smart cities, industrial IoT (Industry 4.0). Sensors, actuators, MQTT protocol. Privacy risks, data collection. 5G connectivity.

Key vocabulary

IoT — Internet of Things: network of connected physical devices. Sensor — detects physical phenomena (temperature, motion). Actuator — a device that performs physical actions. Embedded system — a computer integrated into a device. Smart home — IoT-connected household appliances. Edge computing — processing data near the source, not in the cloud.

16
Software Development & Agile
Software Development Life Cycle (SDLC). Waterfall vs Agile vs Scrum. Sprints, backlogs, stand-ups. Version control: Git. Testing: unit, integration, regression. DevOps.

Key vocabulary

SDLC — Software Development Life Cycle. Agile — iterative development methodology. Sprint — a fixed time box (usually 2 weeks) in Scrum. Git — distributed version control system. Bug — an error in software. Debugging — the process of finding and fixing bugs. Open source — software with publicly available source code.

Society & Employment

Topics 17–20

17
Social Media & Digital Communication
Social media history and platforms. Filter bubbles and echo chambers. Fake news and digital literacy. Screen time and mental health. Digital communication: email, instant messaging, video calls.

Key vocabulary

Algorithm — the system that decides what content you see. Filter bubble — a personalised information environment. Misinformation — false information spread without malicious intent. Disinformation — deliberately false information. Digital footprint — the trace you leave online. Influencer — a person with significant social media following.

18
Privacy, Ethics & GDPR
Data protection principles. GDPR (EU, 2018): rights of data subjects, consent, breach notification. Net neutrality. Intellectual property: copyright, patents. Digital rights.

Key vocabulary

GDPR — General Data Protection Regulation (EU). Personal data — any information relating to an identifiable person. Consent — freely given agreement to data processing. Data breach — unauthorised access to personal data. Right to erasure — "right to be forgotten" under GDPR. Net neutrality — treating all internet traffic equally.
19
Employment in Tech
IT job roles: software engineer, network administrator, data scientist, UX designer, cybersecurity analyst. CVs and cover letters in English. Technical interviews. Remote work and global teams.

Key vocabulary

CV / Résumé — a document summarising professional experience. Cover letter — a letter introducing your application. Technical interview — interview with coding challenges and system design questions. Agile mindset — adaptable, collaborative approach. Soft skills — communication, teamwork, problem-solving. Full-stack — competence in both front-end and back-end development.
20
Emerging Technologies
Blockchain: distributed ledgers, cryptocurrency, smart contracts. Virtual Reality (VR) and Augmented Reality (AR). Quantum computing. 5G and beyond. Ethical issues: automation and jobs.

Key vocabulary

Blockchain — a distributed, immutable ledger of transactions. Cryptocurrency — a digital currency using cryptography. Smart contract — self-executing code on a blockchain. VR — Virtual Reality: fully immersive digital environment. AR — Augmented Reality: digital overlay on the real world. Quantum computing — using quantum mechanics for computation.
