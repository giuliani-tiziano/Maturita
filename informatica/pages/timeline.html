Cronologia del programma

Percorso didattico dell'anno scolastico 2025/2026 — fino al 15 maggio 2026.

Primo Semestre
Set–Ott 2025
Modulo 1 — Basi di Dati
Introduzione ai database e ai DBMS. Sistema informatico vs sistema informativo. Architettura ANSI/SPARC. Struttura della relazione: attributi, tuple, domini, schema e istanza. Grado e cardinalità. Superchiavi, chiavi candidate, chiave primaria e chiave esterna.
Ott–Nov 2025
Modulo 1 (cont.) — Vincoli e Algebra Relazionale
Vincoli di dominio, integrità referenziale, unicità, NOT NULL. Gestione valori NULL. Regole di inserzione, cancellazione (CASCADE, RESTRICT, SET NULL) e modifica. Algebra relazionale: operatori insiemistici (∪ ∩ − ×) e relazionali (σ π ⋈ ρ). Collegamento con SQL.
Nov–Dic 2025
Modulo 2 — PHP lato server
Architettura client/server e ruolo di PHP. phpMyAdmin. Sintassi di base: variabili ($), costanti, tipi di dato, operatori. Scope delle variabili e variabili superglobali ($_GET, $_POST, $_SESSION, $_COOKIE, $_SERVER). Strutture di controllo: if/elseif/else, switch, operatore ternario, for, while, do-while, foreach.
Dic 2025
Modulo 2 (cont.) — Array, Funzioni, Form
Array indicizzati e associativi. Funzioni sugli array. Funzioni personalizzate e anonime. Gestione stringhe, date, formattazione output. Espressioni regolari. Gestione form con GET e POST. Cookie e sessioni. Autenticazione utenti e hashing (password_hash, password_verify). Gestione file. JSON. MySQLi.
Gen 2026
Modulo 3 — JavaScript lato client
Sintassi JS: variabili (var/let/const), array, oggetti, strutture di controllo, funzioni. Gestione eventi e DOM. Manipolazione dinamica degli elementi HTML. Creazione, modifica e rimozione dinamica di contenuti. Tabelle HTML dinamiche. Interazione HTML/CSS/JS.
Secondo Semestre
Feb 2026
Modulo 3 (cont.) — AJAX e JSON
Oggetto XMLHttpRequest. Validazione form lato client. Chiamate asincrone tramite AJAX. JSON per scambio dati (JSON.parse, JSON.stringify). Aggiornamento dinamico senza ricaricamento. Debugging con DevTools.
Feb–Mar 2026
Modulo 4 — SQL
DDL: CREATE, ALTER, DROP, TRUNCATE. DML: INSERT, UPDATE, DELETE. SELECT con WHERE, ORDER BY, GROUP BY, HAVING, LIMIT. Funzioni di aggregazione (COUNT, SUM, AVG). Funzioni MySQL su numeri, stringhe, date (DATE_FORMAT, IF). INNER JOIN, LEFT JOIN, query multi-tabella.
Mar–Apr 2026
Modulo 4 (cont.) — SQL Avanzato
Query annidate e subquery. Operatori ANY, ALL, IN, BETWEEN, DISTINCT, LIMIT, alias. UNION. Viste (VIEW), trigger e stored procedure. INSERT IGNORE e ON DUPLICATE KEY UPDATE. Indici e chiavi per ottimizzazione.
Apr 2026
Modulo 5 — Applicazioni Web Integrate
Integrazione PHP + JavaScript + AJAX + JSON + MySQL. Sviluppo applicazioni web dinamiche. Sicurezza: SQL injection (prepared statements), XSS (htmlspecialchars), session hijacking. CRUD completo. Uso di viste SQL in applicazioni PHP. Temi d'esame.
Apr–Mag 2026
Modulo 6 — Progettazione Concettuale, Logica e Fisica
Progettazione concettuale: analisi requisiti, diagramma E/R, entità/attributi/associazioni, cardinalità, entità deboli, generalizzazioni ISA. Progettazione logica: traduzione E/R→relazionale, dipendenze funzionali, normalizzazione (1NF, 2NF, 3NF). Progettazione fisica: MySQL, tipi di dato, vincoli SQL, phpMyAdmin. Esercitazioni su temi d'esame.
15 Mag 2026
Fine programma svolto
Termine delle attività didattiche. Preparazione all'esame di stato con simulazioni della seconda prova scritta.
