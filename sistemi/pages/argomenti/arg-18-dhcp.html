DHCP
Dynamic Host Configuration Protocol

Assegnazione automatica degli indirizzi IP nella rete.

DHCP è un protocollo client-server che assegna automaticamente indirizzi IP e parametri di rete (subnet mask, default gateway, DNS) agli host. Elimina la necessità di configurazione manuale su ogni dispositivo.

Parametri assegnati da DHCP

  • Indirizzo IP: da un pool (scope)
  • Subnet mask: es. 255.255.255.0
  • Default gateway: IP del router
  • DNS server: IP del server DNS
  • Lease time: durata della concessione
«DHCP è il postino della rete: consegna gli indirizzi senza che nessuno li chieda direttamente.»— Cisco Networking Academy
01

Il Processo DORA

Il processo DHCP usa 4 messaggi broadcast in sequenza, detto DORA:

  1. DHCP Discover: il client invia un broadcast cercando un server DHCP disponibile.
  2. DHCP Offer: il server risponde con un'offerta di indirizzo IP e parametri.
  3. DHCP Request: il client accetta l'offerta e richiede ufficialmente l'indirizzo.
  4. DHCP Acknowledge (ACK): il server conferma l'assegnazione. Il client può ora usare l'IP.

Lease e Rinnovo

La lease è la durata temporanea dell'indirizzo assegnato. Al 50% del lease il client tenta il rinnovo (unicast al server). Al 87.5% prova di nuovo. All'87.5% fallito, cerca un nuovo server con broadcast. Alla scadenza, l'IP torna nel pool.

DHCP Relay Agent

I router non inoltrano i broadcast. In reti con subnet multiple, si configura un DHCP Relay Agent su ogni router: converte i DHCP Discover broadcast in unicast verso il server DHCP centrale. Cisco IOS: ip helper-address [IP_server] sull'interfaccia.

02

Configurazione Cisco IOS

DHCP Server su Cisco IOS

! Escludere IP riservati (gateway, server)
ip dhcp excluded-address 192.168.1.1 192.168.1.10

! Definire lo scope (pool)
ip dhcp pool LAN_POOL
  network 192.168.1.0 255.255.255.0
  default-router 192.168.1.1
  dns-server 8.8.8.8 8.8.4.4
  lease 7              ! 7 giorni

! Verifica
show ip dhcp binding   ! IP assegnati
show ip dhcp pool      ! statistiche pool

DHCP Relay Agent

interface fa0/0
  ip helper-address 10.0.0.1  ! IP del DHCP server
